Mes Travaux

Labyrinthe en python

Ce jeu a été réalisé en python pour le baccalauréat 2019 avec l'aide de Sakina DAVOUST. Son aspect étrange et sombre est un genre de blague entre nous, j'en suis désolée.
Nous avons suivi les tutoriels de TokyoEdTech sur youtube.
Il s'agit d'un simple jeu de labyrinthe dans lequel il faut ramasser tous les trésors dans le temps imparti.
Je vous invite à cliquer sur l'image pour télécharger le projet et à vous amuser. Vous pouvez aisément modifier le jeu via le fichier "map.txt".
Si vous respectez le codage lu par le programme : X est un mur, P le joueur et T les trésors, vous pouvez modifier à l'infini la carte du jeu !
